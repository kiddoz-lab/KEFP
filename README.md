# KEFP

- **KEFP** means **KiddoZ Encrypted File Package**

- <ins>***KEFP is a tool to encrypt and decrypt your files***</ins>, and <ins>entire   folders into one file</ins> its *lightweight* and, the **encryption is, very effective** and, this **software does not break your files**, and you can **encrypt any file format** you want and **decrypt without any damage to your files.**

***

# How it works

## Encryption

- Package: Folders are flattened into a structured stream.

- Compress: Data is shrunk to save space during transfer.

- Encode: Binary data is converted to Base64 for safe transport.

- JSON Wrap: Metadata is added to ensure the package is recognizable by the KiddoZ ecosystem.

## Decryption

- JSON Unwrap: Metadata is removed to reveal the original file structure.

- Decode: Base64 data is converted back to binary.

- Decompress: Data is expanded to its original size.

- Unpackage: The structured stream is converted back into folders.
